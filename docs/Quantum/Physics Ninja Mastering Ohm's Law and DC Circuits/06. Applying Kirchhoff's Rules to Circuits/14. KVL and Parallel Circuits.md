# KVL và Mạch Song Song

## Giới thiệu

Định luật điện áp Kirchhoff (KVL) có thể được áp dụng để phân tích các mạch song song, giúp hiểu rõ hơn về mối quan hệ hiệu điện thế giữa các nhánh.

## Tính chất của mạch song song

### Hiệu điện thế

**Quan trọng:** Trong mạch song song, hiệu điện thế giữa hai điểm nút là **như nhau** cho tất cả các nhánh.

$$V_{AB} = V_1 = V_2 = V_3 = ...$$

### Dòng điện

Dòng điện phân chia tại các nút theo KCL:
$$I_{\text{tổng}} = I_1 + I_2 + I_3 + ...$$

## Ví dụ: Phân tích mạch song song

### Sơ đồ

```
         12V
           │
      ─────┴────
           │
     ┌─────┼─────┐
     │     │     │
   R1┘   R2┘   R3┘
   4Ω    6Ω    12Ω
     │     │     │
     └─────┼─────┘
           │
      ─────┴────
           │
          ─┴─
```

### Phân tích

**Bước 1:** Xác định hiệu điện thế trên mỗi nhánh
- Tất cả các nhánh đều có cùng hiệu điện thế 12 V

**Bước 2:** Tính dòng điện mỗi nhánh (Định luật Ohm)
- $I_1 = V/R_1 = 12/4 = 3$ A
- $I_2 = V/R_2 = 12/6 = 2$ A
- $I_3 = V/R_3 = 12/12 = 1$ A

**Bước 3:** Tính dòng điện tổng (KCL)
$$I_{\text{tổng}} = I_1 + I_2 + I_3 = 3 + 2 + 1 = 6 \text{ A}$$

**Bước 4:** Tính điện trở tương đương
$$R_{eq} = \frac{V}{I_{\text{tổng}}} = \frac{12}{6} = 2 \Omega$$

**Kiểm tra:**
$$\frac{1}{R_{eq}} = \frac{1}{4} + \frac{1}{6} + \frac{1}{12} = \frac{3+2+1}{12} = \frac{6}{12} = \frac{1}{2}$$
$$R_{eq} = 2 \Omega \checkmark$$

## Áp dụng KVL trong mạch song song

### Vòng qua từng nhánh

**Vòng 1 (nguồn → R1 → nguồn):**
$$12 - I_1 R_1 = 0 \Rightarrow 12 - 4I_1 = 0 \Rightarrow I_1 = 3 \text{ A}$$

**Vòng 2 (nguồn → R2 → nguồn):**
$$12 - I_2 R_2 = 0 \Rightarrow 12 - 6I_2 = 0 \Rightarrow I_2 = 2 \text{ A}$$

**Vòng 3 (nguồn → R3 → nguồn):**
$$12 - I_3 R_3 = 0 \Rightarrow 12 - 12I_3 = 0 \Rightarrow I_3 = 1 \text{ A}$$

### Nhận xét

Mỗi vòng trong mạch song song cho phép tính dòng điện nhánh một cách độc lập - đây là ưu điểm của mạch song song!

## Mạch hỗn hợp

### Sơ đồ

```
        R1
    ────□────
         │
        ┌┴┐
        │R│ R2
        └┬┘
         │
    ────□──── R3
         │
        ─┴─
```

### Phân tích

1. R2 song song với R3
2. R1 nối tiếp với (R2 // R3)

**Bước 1:** R2 // R3
$$R_{23} = \frac{R_2 R_3}{R_2 + R_3}$$

**Bước 2:** R1 nối tiếp với $R_{23}$
$$R_{eq} = R_1 + R_{23}$$

## Kết luận

KVL là công cụ hữu ích để phân tích cả mạch nối tiếp và song song. Trong mạch song song:
- Hiệu điện thế như nhau trên mỗi nhánh
- Có thể áp dụng KVL cho từng vòng để tìm dòng điện nhánh độc lập
- Kết hợp với KCL để phân tích hoàn chỉnh mạch
