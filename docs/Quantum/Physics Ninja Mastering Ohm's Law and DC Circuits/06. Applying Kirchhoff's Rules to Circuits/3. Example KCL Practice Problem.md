# Ví Dụ Thực Hành Định Luật Dòng Điện Kirchhoff

## Bài toán 1

### Sơ đồ mạch

```
        I1 = 10A
          │
    ──────┼─────
          │
         ┌┴┐
         │R│  R1 = 5Ω
         └┬┘
          │
    ──────┼─────
          │
         ┌┴┐
         │R│  R2 = 10Ω
         └┬┘
          │
    ──────┼─────
          │
        I2 = ?
```

### Giải

**Bước 1:** Xác định nút
- Có một nút tại điểm nối giữa các nhánh

**Bước 2:** Áp dụng KCL
$$I_1 = I_2$$

**Bước 3:** Kết luận
$$I_2 = 10 \text{ A}$$

## Bài toán 2

### Sơ đồ mạch

```
          I1 = 8A
            │
      ──────┼─────
            │
       ┌────┼────┐
       │    │    │
      I2   I3   I4
       │    │    │
      R1   R2   R3
       │    │    │
       └────┼────┘
            │
      ──────┼─────
            │
            I5
```

Cho biết: $I_1 = 8$ A, $I_2 = 2$ A, $I_3 = 3$ A
Tìm: $I_4$ và $I_5$

### Giải

**Áp dụng KCL tại nút trên:**
$$I_1 = I_2 + I_3 + I_4$$
$$8 = 2 + 3 + I_4$$
$$I_4 = 8 - 5 = 3 \text{ A}$$

**Áp dụng KCL tại nút dưới:**
$$I_2 + I_3 + I_4 = I_5$$
$$2 + 3 + 3 = I_5$$
$$I_5 = 8 \text{ A}$$

### Kiểm tra

$$I_1 = I_5 \Rightarrow 8 = 8 \text{ (đúng)}$$

## Bài toán 3: Mạch hỗn hợp

### Sơ đồ

```
         R1
    ────□────
         │
        ┌┴┐
        │R│ R2
        └┬┘
         │
    ────□──── R3
         │
        ─┴─
```

Cho biết dòng qua R1 là 5 A, qua R3 là 2 A. Tìm dòng qua R2.

### Giải

**Phân tích:**
- Dòng vào nút = Dòng ra khỏi nút
- Tại nút A (nối R1, R2, R3):
$$I_{R1} = I_{R2} + I_{R3}$$

**Thay số:**
$$5 = I_{R2} + 2$$
$$I_{R2} = 3 \text{ A}$$

## Bài toán 4: Xác định chiều dòng điện

### Sơ đồ

```
          I1 = ?
            │
      ──────┼─────
            │
       ┌────┼────┐
       │    │    │
      3A   5A   ?
       │    │    │
      R1   R2   R3
       │    │    │
       └────┼────┘
            │
      ──────┼─────
            │
          I4 = 2A (đi ra)
```

### Giải

**Quy ước:** Dòng đi vào lấy dấu (+), dòng đi ra lấy dấu (-)

**Áp dụng KCL:**
$$I_1 + 3 + 5 - I_4 = 0$$
$$I_1 + 8 - 2 = 0$$
$$I_1 = -6 \text{ A}$$

**Kết luận:**
- Dấu (-) cho thấy chiều thực của $I_1$ ngược với chiều đã vẽ
- Dòng $I_1$ thực tế đi vào nút với độ lớn 6 A

## Lưu ý khi giải bài toán KCL

1. **Chọn chiều dòng điện**: Có thể chọn tùy ý, kết quả âm cho biết chiều ngược lại
2. **Đơn vị**: Chú ý đơn vị của các dòng điện
3. **Kiểm tra**: Luôn kiểm tra lại kết quả bằng cách áp dụng KCL cho các nút khác
4. **Nút điện trở 0**: Hai điểm nối bằng dây dẫn lý tưởng là cùng một nút
