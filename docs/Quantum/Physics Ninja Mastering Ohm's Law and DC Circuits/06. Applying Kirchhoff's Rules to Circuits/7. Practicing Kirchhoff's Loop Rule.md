# Thực Hành Định Luật Vòng Kirchhoff

## Bài toán 1: Mạch đơn giản

### Sơ đồ

```
         12V
           │
      ─────┴────
           │
      ┌────┴────┐
      │    R1   │  4Ω
      └────┬────┘
           │
      ┌────┴────┐
      │    R2   │  6Ω
      └────┬────┘
           │
      ─────┴────
           │
          ─┴─
```

### Giải

**Bước 1:** Chọn chiều dương của vòng (theo chiều kim đồng hồ)

**Bước 2:** Áp dụng KVL

$$12 - I R_1 - I R_2 = 0$$

$$12 - 4I - 6I = 0$$

$$12 - 10I = 0$$

$$I = 1,2 \text{ A}$$

**Bước 3:** Tính hiệu điện thế

- $V_{R1} = I \times R_1 = 1,2 \times 4 = 4,8$ V
- $V_{R2} = I \times R_2 = 1,2 \times 6 = 7,2$ V

**Kiểm tra:** $4,8 + 7,2 = 12$ V ✓

## Bài toán 2: Mạch hai vòng

### Sơ đồ

```
         24V
           │
      ─────┴────
           │
      ┌────┴────┐  R1 = 10Ω
      │    R1   │
      └────┬────┘
           │
      ─────┼────
           │
     ┌─────┴─────┐
     │    R2     │  R2 = 6Ω
     └─────┬─────┘
           │
     ┌─────┴─────┐
     │    R3     │  R3 = 4Ω
     └─────┬─────┘
           │
      ─────┴────
           │
          ─┴─
```

### Giải

**Bước 1:** Mạch gồm R1 nối tiếp với (R2 // R3)

**Bước 2:** Tính R2 // R3

$$R_{23} = \frac{6 \times 4}{6 + 4} = \frac{24}{10} = 2,4 \Omega$$

**Bước 3:** Tính R tổng

$$R_{eq} = R_1 + R_{23} = 10 + 2,4 = 12,4 \Omega$$

**Bước 4:** Tính dòng điện

$$I = \frac{24}{12,4} \approx 1,94 \text{ A}$$

**Bước 5:** Tính hiệu điện thế

- $V_{R1} = I \times 10 = 19,4$ V
- $V_{R2,3} = I \times 2,4 = 4,66$ V

- $I_{R2} = \frac{4,66}{6} \approx 0,78$ A
- $I_{R3} = \frac{4,66}{4} \approx 1,17$ A

## Bài toán 3: Kiểm tra chiều

### Sơ đồ

```
         10V
           │
      ─────┴────
           │
      ┌────┴────┐  R1 = 5Ω
      └────┬────┘
           │
      ┌────┴────┐  R2 = 5Ω
      └────┬────┘
           │
      ─────┴────
           │
         5V
           │
          ─┴─
```

### Giải

**Bước 1:** Chọn chiều vòng ngược kim đồng hồ

**Bước 2:** Áp dụng KVL

$$-10 + I R_1 + I R_2 + 5 = 0$$

$$-10 + 5I + 5I + 5 = 0$$

$$-5 + 10I = 0$$

$$I = 0,5 \text{ A}$$

**Kết quả:** Dòng điện 0,5 A chạy theo chiều đã chọn (ngược kim đồng hồ)

## Lưu ý khi giải

1. **Quy ước dấu nhất quán**: Luôn giữ nguyên quy ước trong suốt bài toán
2. **Chiều dòng điện**: Có thể giả sử chiều, kết quả âm cho biết ngược chiều
3. **Kiểm tra**: Luôn kiểm tra bằng cách tính lại hoặc dùng KVL
4. **Nhiều vòng**: Cần viết phương trình cho mỗi vòng độc lập
