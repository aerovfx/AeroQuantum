# Ký Hiệu Quy Ước trong Sơ Đồ Mạch Điện

## Giới thiệu

Để vẽ và đọc sơ đồ mạch điện một cách thống nhất, người ta sử dụng các **ký hiệu quy ước** chuẩn. Phần này giới thiệu các ký hiệu phổ biến trong mạch điện một chiều.

## Nguồn điện

### Pin/Acquy

```
    ──────┬─────  ← Cực dương (+)
          │
    ──────┴─────  ← Cực âm (-)
    
    Hoặc:
    
    ────|─────────── (Dài: +, Ngắn: -)
    
    Hoặc:
    
         │
    ┌────┴────┐
    │    │    │
    │  ──┼──  │
    └────┼────┘
```

### Nguồn DC có điện trở trong

```
    ┌─────────────────────┐
    │      ╔═══════╗       │
    │      ║   ε   ║       │
 ───┤ ────╢    r    ╟──────┼──
    │      ║       ║       │
    └──────╚═══════╝───────┘
```

## Linh kiện thụ động

### Điện trở

```
    ┌──────┐
────┤      ├────
    └──────┘

    Hoặc (Mỹ):
    
────╰────╯────

    Biến trở (Variabe resistor):
    
────╰────╯────m
```

### Tụ điện

```
    ───┬───
       │
    ───┴───

    Hoặc:
    
────|    |────
       ||
────|    |────
```

### Cuộn cảm

```
    ╭───────╮
───╯       ╰───
    ╰───────╯
```

## Thiết đo

### Ampe kế (Ammeter)

```
    ┌─────────┐
    │    A    │
───►│         │◄──
    └─────────┘
```

### Vôn kế (Voltmeter)

```
    ┌─────────┐
    │    V    │
────┤         ├────
    └─────────┘
```

### Watt kế (Wattmeter)

```
    ┌─────────┐
    │    W    │
────┤         ├────
    └─────────┘
```

## Thiết bị điều khiển

### Công tắc (Switch)

```
    ┌────┐
────┤    │
    └────┘    → Mở (OFF)

    ┌────┐
────┤/   │
    └────┘    → Đóng (ON)
```

### Cầu dao (Circuit breaker)

```
────╬────    → Đóng

────╫────    → Mở
```

### Relay

```
    ┌───────┐
    │   ◯   │
────┤   │   ├────
    └───┼───┘
        │
    ────┴────
```

## Thiết bị sử dụng điện

### Bóng đèn

```
    ╭───╮
───╯     ╰───
    ╰───╯

    Hoặc:
    
────✰────
```

### Động cơ

```
    ╭─────╮
───╯ ⓜ   ╰───
    ╰─────╯
```

### Diode

```
────►│────

    Anode (A) ─►│─── Katode (K)
```

## Các ký hiệu khác

### Đất (Ground)

```
────┴────    hoặc    ═══

```

### Không nối đất (No connection)

```
○    hoặc    NC
```

### Điểm nối (Connection dot)

    │
────┼────
    │
```

         ┌─────────┐

 ┌──┴──┐              ┌┴──┐

```

## Kết luận

Việc nắm vững các ký hiệu quy ước trong sơ đồ mạch điện là kỹ năng cơ bản giúp đọc và vẽ sơ đồ mạch chính xác, thuận tiện cho việc phân tích và thiết kế mạch điện.
