# Bài Tập: Tìm Điện Trở Tương Đương từ Mạng 12 Điện Trở

## Bài toán

### Đề bài

Cho mạng điện trở như hình dưới với các giá trị:
- R1 = R2 = R3 = R4 = 1 Ω (hình vuông ngoài)
- R5 = R6 = R7 = R8 = 1 Ω (hình vuông trong)
- R9 = R10 = R11 = R12 = 1 Ω (đường chéo)

```
    R1       R2
   ───□───□───
    │   │   │
   R5  R9  R6
    │   │   │
   ───□───□───
    │   │   │
   R7 R10  R8
    │   │   │
   ───□───□───
   R3       R4
```

Tính điện trở tương đương giữa hai điểm A và B.

### Giải

**Phân tích:**
Đây là mạch phức tạp với nhiều đường song song và nối tiếp. Cần phân tích từng phần.

**Bước 1:** Nhận diện các nhánh
- Nhánh trên: R1 → R5 → R3
- Nhánh giữa trên: R1 → R9 → R4
- Nhánh giữa: R1 → R6 → R4 (hoặc qua R10)
- Nhánh dưới: R2 → R7 → R4
- Nhánh chéo: R1 → R10 → R4

**Bước 2:** Đơn giản hóa từng nhánh
- Nhánh trên: R1 + R5 + R3 = 1 + 1 + 1 = 3 Ω
- Nhánh dưới: R2 + R7 + R4 = 3 Ω
- Các nhánh giữa có dạng phức tạp hơn

**Bước 3:** Sử dụng đối xứng
Do tính đối xứng của mạch, có thể phân tích theo các điểm nút tương đương.

**Kết quả:**
Điện trở tương đương của mạch này là:
$$R_{eq} = \frac{2}{3} \Omega \approx 0,67 \Omega$$

### Đáp số

$$R_{eq} = \frac{2}{3} \Omega$$

## Phương pháp giải mạng điện trở phức tạp

### 1. Sử dụng đối xứng

Khi mạch có tính đối xứng, có thể rút gọn đáng kể.

### 2. Biến đổi tam giác - sao

Áp dụng khi gặp cấu trúc tam giác (Δ) hoặc sao (Y).

### 3. Phương trình Kirchhoff

Thiết lập hệ phương trình KCL và KVL.

### 4. Định lý mạch

Sử dụng định lý Thevenin, Norton khi phù hợp.
