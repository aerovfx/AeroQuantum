# Bài Tập: Xác Định Loại Kết Hợp Điện Trở

## Bài toán 1

### Đề bài

Xác định xem các điện trở trong sơ đồ sau mắc nối tiếp hay song song:

```
    R1         R2
 ───□───────□───
     │
     │
   R3
```

### Giải

**Phân tích:**
- Dòng điện từ nguồn đi qua R1, sau đó chia thành hai nhánh
- Một nhánh đi qua R3, nhánh kia đi qua R2
- R1 mắc **nối tiếp** với nhánh (R2 // R3)
- R2 mắc **song song** với R3

### Kết luận

- R1: Nối tiếp với mạch còn lại
- R2 // R3: Song song với nhau, sau đó nối tiếp với R1

## Bài toán 2

### Đề bài

Xác định loại kết hợp:

```
      R1
      │
 ─────┴────
      │
    ┌─┴─┐
    │R2 │
    └─┬─┘
      │
    ┌─┴─┐
    │R3 │
    └─┬─┘
      │
 ─────┴────
      │
     R4
```

### Giải

**Phân tích:**
- R2 và R3 có chung hai điểm đầu → **song song**
- (R2 // R3) nối tiếp với R1 và R4

### Kết luận

- R2 // R3
- R1 nối tiếp (R2 // R3) nối tiếp R4

## Bài toán 3

### Đề bài

Xác định loại kết hợp:

```
    R1    R2
 ───□───□───
       │
     R3
       │
 ───□──□───
    R4  R5
```

### Giải

**Phân tích:**
- R1 nối tiếp với R2
- R4 nối tiếp với R5
- Nhánh trên (R1-R2) song song với nhánh dưới (R4-R5)
- R3 nối tiếp với cả hai nhánh

### Kết luận

- (R1 // R2) nối tiếp với R3 nối tiếp với (R4 // R5)

---

## Bài toán 4

### Đề bài

Xác định loại kết hợp:

```
      R1
      │
  ────┼────
      │
   ┌──┼──┐
   │  │  │
  R2  R3 R4
   │  │  │
   └──┼──┘
      │
  ────┼────
      │
     R5
```

### Giải

**Phân tích:**
- R2, R3, R4 có chung hai điểm đầu → **song song**
- R1 nối tiếp với nhóm (R2 // R3 // R4)
- R5 nối tiếp với nhóm trên

### Kết luận

- R1 nối tiếp (R2 // R3 // R4) nối tiếp R5

---

## Phương pháp nhận biết

### Nối tiếp
- Cùng một dòng điện chạy qua
- Chỉ một đường duy nhất

### Song song
- Có chung hai điểm đầu
- Có điểm rẽ nhánh
- Hiệu điện thế như nhau trên các thành phần
