# Bài Tập: Kết Hợp Các Kết Hợp

## Bài toán 1

### Đề bài

Cho mạch điện:
- $R_1 = 4$ Ω, $R_2 = 6$ Ω mắc song song
- Nhóm này mắc nối tiếp với $R_3 = 5$ Ω

Tính điện trở tương đương.

### Giải

**Bước 1:** Tính $R_1$ // $R_2$

$$R_{12} = \frac{4 \times 6}{4 + 6} = \frac{24}{10} = 2,4 \Omega$$

**Bước 2:** Tính $R_{eq}$

$$R_{eq} = R_{12} + R_3 = 2,4 + 5 = 7,4 \Omega$$

### Đáp số

$R_{eq} = 7,4$ Ω

## Bài toán 2

### Đề bài

Cho mạch điện:
- $R_1 = 10$ Ω mắc nối tiếp với nhánh ($R_2 = 20$ Ω // $R_3 = 30$ Ω)
- Điện trở này mắc song song với $R_4 = 15$ Ω

Tính điện trở tương đương.

### Giải

**Bước 1:** Tính $R_2$ // $R_3$

$$R_{23} = \frac{20 \times 30}{20 + 30} = \frac{600}{50} = 12 \Omega$$

**Bước 2:** Tính ($R_1$ + $R_{23}$)

$$R_{123} = R_1 + R_{23} = 10 + 12 = 22 \Omega$$

**Bước 3:** Tính $R_{eq}$

$$R_{eq} = \frac{22 \times 15}{22 + 15} = \frac{330}{37} \approx 8,92 \Omega$$

### Đáp số

$R_{eq} \approx 8,92$ Ω

## Bài toán 3: Mạch phức tạp

### Đề bài

```
    R1=10Ω
      │
 ─────┼────
      │
     ┌┴┐ R2=20Ω
     └┬┘
      │
 ─────┼────
      │
     ┌┴┐ R3=30Ω
     └┬┘
      │
 ─────┼────
     R4=15Ω
       │
      ─┴─
```

Tính điện trở tương đương.

### Giải

**Bước 1:** R2 song song R3

$$R_{23} = \frac{20 \times 30}{20 + 30} = \frac{600}{50} = 12 \Omega$$

**Bước 2:** Mạch trở thành: R1 nối tiếp R23 nối tiếp R4

$$R_{eq} = R_1 + R_{23} + R_4 = 10 + 12 + 15 = 37 \Omega$$

### Đáp số

$R_{eq} = 37$ Ω

## Bài toán 4

### Đề bài

```
    R1=8Ω        R4=6Ω
      │           │
 ─────┼───────┬───┼────
      │       │   │
     ┌┴┐     ┌┴┐ ┌┴┐
     │R│     │R│ │R│
     │2│     │3│ │5│
     │5Ω│     │6Ω│ │12Ω│
     └┬┘     └┬┘ └┬┘
      │       │   │
 ─────┼───────┼───┼────
      │       │   │
      └───────┴───┘
```

Tính điện trở tương đương.

### Giải

**Bước 1:** R3 song song R5

$$R_{35} = \frac{6 \times 12}{6 + 12} = \frac{72}{18} = 4 \Omega$$

**Bước 2:** R1 song song với (R2 + R4) // R35

- R2 nối tiếp R4: $R_{24} = 5 + 6 = 11$ Ω
- (R2+R4) song song R35: $R_{2435} = \frac{11 \times 4}{11 + 4} = \frac{44}{15} \approx 2,93$ Ω

**Bước 3:** R1 song song với $R_{2435}$

$$R_{eq} = \frac{8 \times 2,93}{8 + 2,93} = \frac{23,44}{10,93} \approx 2,15 \Omega$$

### Đáp số

$R_{eq} \approx 2,15$ Ω

## Lưu ý khi giải mạch phức tạp

1. **Vẽ lại sơ đồ** cho dễ nhìn
2. **Xác định rõ** các cặp nối tiếp/song song
3. **Giải từng bước** một
4. **Luôn kiểm tra** kết quả
