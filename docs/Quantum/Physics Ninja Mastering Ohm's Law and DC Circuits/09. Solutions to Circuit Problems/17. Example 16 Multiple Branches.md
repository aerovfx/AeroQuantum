# Bài Tập Nhiều Nhánh

## Bài toán: Phân tích mạch nhiều nhánh

### Đề bài

```
         24V
           │
      ┌────┴────┐
      │    R1    │  6Ω
      └────┬────┘
           │
     ┌─────┼─────┐
     │     │     │
   R2┘   R3┘   R4┘
   4Ω    8Ω   12Ω
     │     │     │
     └─────┼─────┘
           │
         ─┴─
```

### Giải

**Bước 1:** Tính điện trở tương đương của 3 nhánh song song

$$\frac{1}{R_{123}} = \frac{1}{4} + \frac{1}{8} + \frac{1}{12} = \frac{6+3+2}{24} = \frac{11}{24}$$

$$R_{123} = \frac{24}{11} \approx 2,18 \Omega$$

**Bước 2:** Tính điện trở tổng

$$R_{eq} = R_1 + R_{123} = 6 + 2,18 = 8,18 \Omega$$

**Bước 3:** Tính dòng điện tổng

$$I = \frac{24}{8,18} \approx 2,93 \text{ A}$$

**Bước 4:** Tính dòng điện từng nhánh

- $I_1 = 2,93$ A (qua R1)
- $I_2 = V/R_2 = 24/4 = 6$ A ← Cần kiểm tra lại vì tổng dòng nhánh > dòng tổng!

**Lưu ý:** Cần xem lại mô hình mạch. Đây là mạch 4 nhánh song song.
