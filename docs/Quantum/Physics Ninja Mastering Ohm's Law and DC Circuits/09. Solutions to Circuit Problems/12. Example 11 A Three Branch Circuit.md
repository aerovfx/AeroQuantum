# Bài Tập Mạch Ba Nhánh

## Bài toán

### Đề bài

```
        12V
          │
     ┌────┴────┐
     │    R1    │  4Ω
     └────┬────┘
          │
    ┌─────┼─────┐
    │     │     │
  R2┘   R3┘   R4┘
  6Ω    8Ω   12Ω
    │     │     │
    └─────┼─────┘
          │
        ─┴─
```

Tính dòng điện qua mỗi nhánh.

### Giải

**Bước 1:** Tính điện trở tương đương

$$R_{eq} = 4 + \frac{1}{\frac{1}{6} + \frac{1}{8} + \frac{1}{12}} = 4 + 2 = 6 \Omega$$

**Bước 2:** Tính dòng điện tổng

$$I = \frac{12}{6} = 2 \text{ A}$$

**Bước 3:** Tính dòng điện từng nhánh

- $I_1 = 2$ A (qua R1)
- $I_2 = V/R_2 = 12/6 = 2$ A
- $I_3 = V/R_3 = 12/8 = 1,5$ A
- $I_4 = V/R_4 = 12/12 = 1$ A

**Kiểm tra:** $I_2 + I_3 + I_4 = 2 + 1,5 + 1 = 4,5$ A ≠ 2 A

→ Cần xem lại cách mắc! Đây là mạch 4 nhánh song song sau R1
