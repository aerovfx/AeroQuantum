# Bài Tập Mạch Điện

## Bài toán 1: Tìm dòng điện

### Đề bài

Cho mạch điện như hình với các giá trị:
- Nguồn: 18V
- R1 = 3Ω, R2 = 6Ω, R3 = 9Ω

```
      R1
       │
   ────┼────
       │
     R2
       │
   ────┼────
       │
     R3
       │
     ─┴─
```

### Giải

$$R_{eq} = 3 + 6 +18 \Omega$$

 9 = $$I = \frac{18}{18} = 1 \text{ A}$$

## Bài toán 2: Phân tích mạch hỗn hợp

### Đề bài

```
    12V
      │
  ┌───┴───┐
  │   R1   │  4Ω
  └───┬───┘
      │
  ┌───┴───┐
  │   R2   │  6Ω
  └───┬───┘
      │
    ──┴──
```

Tính dòng điện và công suất.

### Giải

$$R_{eq} = 4 + 6 = 10 \Omega$$

$$I = \frac{12}{10} = 1,2 \text{ A}$$

$$P = I^2 R_{eq} = 1,2^2 \times 10 = 14,4 \text{ W}$$

## Bài toán 3: Mạch với nhiều nguồn

### Đề bài

```
    10V    R1=5Ω    5V
      │──────□─────│
                  │
                R2=10Ω
                  │
      │──────□─────│
      │            │
      └────────────┘
```

### Giải

**Phương trình KVL:**
$$10 - 5I - 5 = 0$$
$$5 - 5I = 0$$
$$I = 1 \text{ A}$$
